Paso 1️⃣: CREAR BASE DE DATOS
   Abre MySQL y ejecuta:
   CREATE DATABASE desafio2db;

Paso 2️⃣: INICIAR BACKEND
   
   OPCIÓN A (Windows):
   └─ Haz doble clic en INICIAR.bat
   
   OPCIÓN B (Linux/Mac):
   └─ chmod +x INICIAR.sh
   └─ ./INICIAR.sh
   
   OPCIÓN C (Manual - Cualquier SO):
   └─ mvnw clean package
   └─ mvnw spring-boot:run

   ✅ El backend estará en http://localhost:8081

Paso 3️⃣: ABRIR FRONTEND
   └─ Navega a la carpeta frontend
   └─ Abre index.html en tu navegador
   
   O con Python:
   └─ cd frontend
   └─ python -m http.server 8000
   └─ Abre http://localhost:8000

P: ¿Cómo cambio la URL del backend?
R: Abre frontend/js/api.js y busca "const API_BASE_URL"

P: ¿Olvidé la contraseña?
R: Crea una nueva cuenta con "Registrarse"

P: ¿Dónde están los datos guardados?
R: En la base de datos MySQL (desafio2db)

P: ¿Puedo usar este proyecto en producción?
R: No directamente. Necesita configuración de seguridad adicional.

P: ¿Cómo agrego más campos a una entidad?
R: Edita la clase en domain/, crea una migración SQL, y actualiza el frontend.
